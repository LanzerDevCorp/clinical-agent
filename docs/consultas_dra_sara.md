## 1. Confirmación de la Tipología Estática de Productos (`productType`)

Para organizar el catálogo en el CMS y optimizar las búsquedas del agente de chat, necesitamos validar si esta clasificación de productos es correcta o si requiere agregar/modificar alguna categoría:

| Tipo de Producto (Propuesto) | Descripción | Ejemplos | ¿Es Correcto? (Sí / No / Observaciones) |
| :--- | :--- | :--- | :--- |
| **Liofilizado** | Inyectable en polvo seco deshidratado al vacío. Requiere reconstitución obligatoria con diluyente. | Bellatox, Wiztox | |
| **Líquido** | Inyectable en solución líquida lista para usar o con dilución simple complementaria. | Sofiderm, Celosome | |
| **Hilos PDO** | Hilos tensores de polidioxanona. | Hilos PDO Mono, COG | |
| **Dispositivo Médico** | Equipos mecánicos o electrónicos de soporte en la aplicación. | Dermapen, Glowpen | |
| **Insumo de Aplicación** | Consumibles estériles desechables de aplicación. | Agujas (30G, 32G), Cánulas | |
